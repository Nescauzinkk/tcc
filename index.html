Projeto de TCC - Site de Estudos para CEEP

Este projeto consiste na criação de um site de estudos voltado para estudantes dos 3 anos de um curso técnico em Desenvolvimento de Sistemas. O objetivo principal é fornecer uma plataforma online com conteúdo relevante para as matérias específicas do curso, como lógica de programação, banco de dados, algoritmos, redes de computadores e outras disciplinas voltadas para o desenvolvimento de sistemas.

Matérias

  • Front-end: HTML, CSS, JavaScript (com uso de frameworks como React.js)
  • Back-end: Node.js, PHP, Python
  • Banco de Dados: MySQL (SQL)
  • Controle de Versão: Git/GitHub
  • COmputação Gráfica: CorelDraw, Photoshop, Canva

Objetivos do Projeto

  • Criar uma plataforma de estudos online com conteúdo exclusivo para alunos de cursos técnicos em Desenvolvimento de Sistemas.
  • Desenvolver funcionalidades como: criação de contas de usuários, login/logout, busca de conteúdo e visualização de material didático.
  • Prover materiais interativos como quizzes, tutoriais em vídeo e exercícios práticos.
  • Implementar um sistema de avaliações e acompanhamento do progresso do aluno.

Funcionalidades

1. Cadastro e Login de Usuários

  • Os alunos podem criar suas contas e acessar o site de forma personalizada.
  • A autenticação é realizada por meio de e-mail e senha.
  • O sistema utiliza JWT para garantir a segurança na comunicação.

2. Módulos de Estudo

  • Conteúdo de diversas matérias do curso de Desenvolvimento de Sistemas:
    • Lógica de Programação
    • Algoritmos e Estruturas de Dados
    • Banco de Dados (SQL e NoSQL)
    • Redes de Computadores
    • Programação Orientada a Objetos
  • Cada módulo possui textos explicativos, exercícios práticos e quizzes.

3. Sistema de Progresso

  • Acompanhamento do desempenho do aluno em cada módulo, com relatórios gráficos.
  • Feedback personalizado baseado nos acertos e erros nos quizzes.

4. Busca e Filtro de Conteúdo

  • Os alunos podem buscar por termos específicos ou filtrar conteúdos por matéria ou tipo (texto, vídeo, quiz, etc.).

5. Área de Admin

  • O administrador pode adicionar, editar ou remover materiais didáticos, além de gerenciar usuários.
  • O administrador tem acesso aos relatórios de progresso de todos os alunos.

Instalação

Pré-requisitos

  • Node.js (versão 14.x ou superior)
  • npm (gerenciador de pacotes do Node.js)
  • MongoDB (para o banco de dados NoSQL) ou MySQL (para banco de dados relacional)
  • Git (para controle de versão)

Passos para rodar o projeto

  1. Clone o repositório: 
    git clone https://github.com/usuario/tcc-site-estudos.git
  2. Navegue até o diretório do projeto: 
    cd tcc-site-estudos
  3. Instale as dependências do backend: 
    cd backend
npm install
  4. Instale as dependências do frontend: 
    cd frontend
npm install
  5. Configure o banco de dados:
    • Se estiver usando MongoDB, configure a conexão no arquivo .env com a URL do seu banco de dados.
    • Se estiver usando MySQL, configure as credenciais no arquivo de configuração.
  6. Inicie o servidor backend: 
    cd backend
npm start
  7. Inicie o frontend: 
    cd frontend
npm start
  8. Acesse o site no navegador em http://localhost:3000.
